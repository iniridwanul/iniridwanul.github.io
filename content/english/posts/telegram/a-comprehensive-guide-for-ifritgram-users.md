## How Ifritgram Works?
Ifritgram function by interacting with the Telegram Bot API, a set of HTTP-based interfaces that allow developers to create bots and integrate them into their applications. When a user sends a message to a userbot, the bot's server receives the message, processes it, and then sends an appropriate response back to the user.

## What is Telethon?
Telethon MTProto is a powerful and versatile Python library designed to interact with the Telegram messaging platform. It provides a user-friendly interface to access the Telegram API, allowing developers to build a wide range of applications, from simple bots to complex chat clients.
